Instructions to Run the Program

1. Download the zip file
2. Extract the zip file and open the folder in Visual Studio Code
3. Open the terminal and run the following command to install the dependencies
  pip install -r requirements.txt
4. Run the following command to run the program
    python main.py
5. The program Interface will open click on upload Image and select the image you want to upload
    Image can be found in Test Images folder
6. The image will be uploaded and the output will be displayed on the screen
